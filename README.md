# Ageha Editor

![Icon](https://ageha-editor.pages.dev/icon-128.png)

## アプリケーション概要

**Ageha Editor**は [Tauri](https://v2.tauri.app/ja/) 製のマークダウンのエディター。軽量かつインストール不要で使用可能。

![Ageha Editor](https://ageha-editor.pages.dev/ageha-editor.png)

### 基本機能

- リアルタイムプレビュー
- 印刷機能（PDF出力を含む）
- 入力支援機能
- フローチャート対応（mermaid）
- 数式対応（KaTeX）
